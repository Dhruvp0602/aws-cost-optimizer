## 📌 Overview

**AWS Cost Optimizer** is a full-stack, serverless-first SaaS application designed to analyze AWS infrastructure costs across multiple regions and accounts.

It leverages **Generative AI (Llama 3.1 via Groq)** along with **AWS STS** to detect unused ("zombie") resources and provide intelligent cost-saving recommendations in real time.

---

## 🚀 Features

### 🌍 Multi-Region Scanning
- Scans **13 AWS regions in parallel**
- Uses `Promise.allSettled` for efficient execution

### 🔗 Cross-Account Access
- Secure integration using **AWS STS AssumeRole**
- Enables **multi-account cost visibility**

### 🤖 AI-Powered Optimization
- Uses **Llama 3.1 (Groq API)**  
- Generates:
  - Cost-saving recommendations  
  - Architecture improvements  
  - Resource optimization insights  

### 📊 Real-Time Dashboard
- Displays:
  - Total potential savings (₹)
  - Resource distribution
  - Service-level cost insights

### 🔍 Resource Monitoring
Tracks:
- EC2 Instances  
- EBS Volumes  
- Snapshots  
- NAT Gateways  
- Elastic IPs  
- Load Balancers

## ⚙️ Technical Workflow

### 1️⃣ Authentication
- User provides **AWS Role ARN**
- Managed via **React Context API**

### 2️⃣ Resource Discovery
- Frontend calls AWS services via:
  - `@aws-sdk/client-ec2`
  - `@aws-sdk/client-sts`
  - `@aws-sdk/client-elastic-load-balancing-v2`

### 3️⃣ Data Refinement
- Filters:
  - Stopped EC2 instances  
  - Unattached EBS volumes  
  - Idle Elastic IPs  
  - Unused Load Balancers  

### 4️⃣ AI Analysis
- Sends structured data to **Groq API**
- Llama 3.1 generates insights

### 5️⃣ Reporting Layer
- Displays recommendations in:
  - Dashboard cards  
  - JSON insights panel
